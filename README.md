# Sun Proactive — Интеллектуальная экосистема для волонтеров ☀️

**Sun Proactive** — это платформа, которая использует ИИ для автоматизации создания задач и точного подбора волонтеров на основе их навыков и доступности.

## Демо-версия доступна по ссылке: (https://img.shields.io/badge/Live-Demo-orange)](https://github.com/Adiya-32/VolunteerHub)

## 🌟 Основные возможности

- **🤖 ЖИ Тапсырма жасаушы:** ИИ-ассистент «Арай», который превращает свободные запросы координаторов в структурированные задания.
- **🧠 Смарт-матчинг:** Алгоритм ИИ, обеспечивающий идеальное соответствие волонтеров требованиям задач (Match Rate до 100%).
- **📋 Тапсырмалар тақтасы:** Визуальное управление статусами и деталями всех волонтерских инициатив.
- **👥 Еріктілер каталогы:** База данных волонтеров с фильтрацией по компетенциям (Marketing, Teaching, First Aid и др.).
- **📝 Еріктілер тіркелімі:** Удобная форма регистрации для новых участников с указанием навыков и графика.

## 🛠️ Технологический стек

- **Frontend:** React 19 (Vite), Tailwind CSS v4, Motion, Lucide Icons.
- **Backend:** Node.js 24, Express 5.
- **Data:** PostgreSQL, Drizzle ORM, Zod.
- **AI:** Gemini AI — для обработки естественного языка и аналитики.

## 🚀 Быстрый старт

### Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Adiya-32/VolunteerHub.git
   ```
2. Перейдите в папку фронтенда:
   ```bash
   cd artifacts/sun-proactive
   ```
3. Установите зависимости и запустите проект:
   ```bash
   npm install
   npm run dev
   ```

## 📂 Структура проекта

- `artifacts/sun-proactive/` — Основное Frontend-приложение.
- `artifacts/api-server/` — Серверная часть и API логика.
- `lib/` — Общие библиотеки и схемы данных.
- `src/services/` — Интеграция с Gemini AI и внешними API.
